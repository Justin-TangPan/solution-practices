Headroom Technology · 核心技术
07 / 10
CORE ENGINE · 上下文压缩引擎

Headroom 四大能力

01

智能压缩

6 种压缩算法：SmartCrusher(JSON)、CodeCompressor(AST 感知)、Kompress-base(ML 模型)、CacheAligner(KV 缓存优化)等，综合节省 60-95% Token。

02

CCR 可逆压缩

Context Compression Reversible 技术——原始数据永不丢失，LLM 可按需检索完整上下文，压缩无感、准确率保持不变。

03

多模式部署

支持 Library(Python/TS)、Proxy(零代码改动)、Agent Wrap(一行命令接入 Claude/Codex/Cursor)、MCP Server 四种模式。

04

跨 Agent 记忆

共享记忆存储在 Claude、Codex、Gemini 之间同步，自动去重。headroom learn 从失败会话中学习并写入 CLAUDE.md。

Deployment · 一键部署上云

HUAWEI CLOUD RFS · 资源编排服务

三步完成云上部署

HOW IT WORKS

SAC 生成的标准化包通过华为云 RFS 一键部署，全程可视化控制台操作，无需手动运维。

01

上传 OBS

Terraform 脚本上传到 OBS 桶（tp-00940108），Shell 安装脚本同步上传，确保路径一致。

02

RFS 创建堆栈

在华为云 RFS 控制台创建堆栈，粘贴 OBS 模板 URL，配置参数后一键部署。自动创建 VPC、ECS、EIP 等资源。

03

自动安装启动

ECS 创建后 user_data 自动执行安装脚本，完成系统准备、运行时安装、应用配置、服务启动全流程。
