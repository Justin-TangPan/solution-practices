## 04. 准备工作

### 页面结构

```
面包屑导航：文档首页 → {分类} → 快速部署{方案名} → 实施步骤 → 准备工作

# 准备工作

## 创建 rf_admin_trust 委托（可选）

> 如果使用新注册的华为云账号，可跳过准备工作直接进入快速部署。
> 如果使用 IAM 子用户，请确认该用户已加入 admin 用户组。

### 操作步骤

1. 登录华为云管理控制台 → 统一身份认证 → 委托
2. 搜索 `rf_admin_trust`
   - 如果已存在 → 跳过以下步骤
   - 如果不存在 → 继续执行步骤 3
3. 单击"创建委托"，委托名称输入 `rf_admin_trust`，委托类型选择"云服务"，输入 `RFS`，单击"完成"
4. 单击"立即授权"
5. 在搜索框中输入 `Tenant Administrator`，勾选搜索结果，单击"下一步"
6. 选择"所有资源"，单击"确定"完成配置
7. 验证：委托列表中出现 `rf_admin_trust` 即为创建成功

---
导航：父主题 → 实施步骤 | 上一篇 → 实施步骤 | 下一篇 → 快速部署
```

### 写作规范

- **步骤编号 1-7**，每步一个操作
- **包含截图**（图1-图8），展示关键操作界面
- **标注可选**：如果新账号可跳过

### 步骤一：进入部署界面

1. 登录华为云[解决方案实践](https://solution.huaweicloud.com/)

3. （包年包月模式）如余额不足，需在费用中心手动完成支付

### 步骤四：等待完成

1. 在事件日志中等待出现"Apply required resource success"
2. 表示资源栈部署成功
3. 部署脚本将自动执行，等待约 {X} 分钟（视网络环境而定）

---
导航：父主题 → 实施步骤 | 上一篇 → 准备工作 | 下一篇 → 开始使用

## 写作总规范

### 文档风格

1. **简洁明了**：每段不超过 3 句话
2. **步骤清晰**：用编号列表，每步一个操作
3. **图文并茂**：关键步骤必须有截图
4. **客户视角**：用"您"而非"用户"

### 术语规范

| 推荐用语 | 避免用语 |
|---------|---------|
| 您 | 用户、我们 |
| 单击 | 点击 |
| 登录 | 登陆 |
| 云服务器 | 虚拟机 |
| 弹性公网 IP | 公网 IP |

### 截图要求

- **格式**：PNG，分辨率 ≥ 1280×720
- **标注**：关键操作用红框/箭头标注
- **脱敏**：隐藏账号、密码、API Key 等敏感信息
- **命名**：`图{X}-{描述}`，如 `图1-解决方案实践库`

### 链接规范

- **内部链接**：使用相对路径，如 `{slug}_05.html`
- **外部链接**：使用完整 URL，如 `https://console.huaweicloud.com/rfs/`
- **华为云控制台**：必须带 region 参数，如 `?region=cn-north-4`
