## 写作规范

### 文档风格

1. **简洁明了**：每段不超过 3 句话
2. **步骤清晰**：用编号列表，每步一个操作
3. **图文并茂**：关键步骤必须有截图
4. **客户视角**：用"您"而非"用户"

### 截图要求

- **格式**：PNG，分辨率 ≥ 1280×720
- **标注**：关键操作用红框/箭头标注
- **脱敏**：隐藏账号、密码、API Key 等敏感信息
- **编号**：`图 {章节}-{序号}`，如 `图 1-1 单机版方案架构图`

### 表格规范

- 资源和成本规划表必须包含两种计费模式（按需 + 包月）
- 参数配置说明表必须包含：参数、说明、默认值 三列
- 所有费用必须加免责声明："预估费用仅供参考，实际费用取决于具体使用情况"

### 模板来源

本模板基于 `LiteLLM，统一的AI管理网关` 部署指南提取，遵循华为云解决方案实践部署指南标准格式。
