# 华为云解决方案实践发布标准

## 1. 架构图规范

| 规范项 | 要求 |
|--------|------|
| 逻辑组网 | 体现资源间逻辑组网关系和业务流转方向 |
| 云服务图标 | 必须使用华为云官方图标 |
| 自建服务 | 使用第三方图标，标注"XX自建服务" |
| 服务器命名 | 体现用途（如"web-server"、"db-server"），不使用默认"弹性云服务器" |
| 资源清单 | 种类数量须与架构图一致 |

## 2. 部署架构类型

| 类型 | 适用场景 | 说明 |
|------|---------|------|
| 手工部署 | 简单场景 | 手动配置规格 |
| 手工部署（图形模式） | 可视化配置 | 拖拽图标配置 |
| 自动部署（RFS） | Terraform 模板 | 一键部署资源 |
| 自动部署（CAM） | 广州区域 | 需要评估是否满足需求 |

## 3. 资源清单配置要求

- 在保证正常运行前提下选择**最低配置**
- 资源清单种类、数量须与架构图**完全一致**
- 所有华为云服务都需配置清单
- 非华为云服务可不配置

## 4. 发布参数

| 参数 | 说明 |
|------|------|
| 模板名称 | 自定义，作为发布名称 |
| 适用行业 | 最多 10 个行业 |
| 适用场景 | 最多 10 个场景 |
| 部署环境 | 与方案所属底座有关 |
| 发布范围 | 企业内部 / 仅华为 / 指定人 / 所有人 |
| 作者标签 | 如"xx行业架构师" |
| 模板描述 | 建议包含业务痛点、方案描述、优势价值、架构说明 |

```markdown

{一句话方案概述}

{架构图}

{架构描述：数据流/业务流说明}

| 资源 | 规格 | 数量 |
|------|------|------|
| {资源名称} | {规格} | {数量} |

### 前置条件

- 已有华为云账号，且账户余额充足
- 已了解{关键概念/服务}

1. 进入{购买/部署页面}
2. 配置{关键参数}

4. 等待部署完成（约{时间}）

### 参数说明

| 参数 | 说明 | 示例 |
|------|------|------|
| {参数名} | {说明} | {示例值} |

{部署完成后如何访问和使用}

## 预估费用

| 资源 | 规格 | 预估月费 |
|------|------|---------|

2. 找到对应资源栈
3. 单击"删除"→ 输入 "Delete" → 确认
```

## 6. 质量标准

- [ ] 架构图使用华为云官方图标
- [ ] 资源规格选择最低可用配置
- [ ] 文档包含前置条件、部署步骤、参数说明
- [ ] 提供预估费用参考
- [ ] 包含快速卸载步骤
- [ ] 服务器命名体现用途
