## 1. 方案概述

### 1.1 应用场景

本方案通过华为云 Flexus 云服务器 X 实例上一键部署 Headroom + OpenCode：一个开源、低成本、终端原生的 AI 编程辅助环境。

该方案通常应用到以下场景：

- **日常开发辅助**：在终端中使用自然语言进行代码编写、重构、调试，Headroom 自动压缩上下文降低 Token 消耗
- **大型代码库探索**：快速阅读和理解不熟悉的代码库，AST 感知压缩对文件内容可达到 88% 的压缩率
- **日志分析与调试**：利用 AI 分析大量日志文件，Headroom 对日志内容的压缩率可达 90-95%
- **架构设计与评审**：在终端中完成系统设计讨论和代码审查，支持切换 DeepSeek、Claude 等多种模型

### 1.2 方案架构

#### 单机版部署

图 1-1 单机版方案架构图

该解决方案将会部署如下资源：

- 创建 1 台华为 Flexus 云服务器 X 实例，用于部署 Headroom 代理和 OpenCode CLI
- 创建 1 个弹性公网 IP EIP 并关联云服务器，提供访问公网和被公网访问能力
- 创建 1 个 VPC 及子网，用于网络隔离

#### 架构流程

1. 开发者在 OpenCode CLI 中输入提示或命令
2. OpenCode 将请求发送至 `http://localhost:8787`（Headroom 代理）
3. Headroom 分析上下文载荷，压缩冗余数据（工具输出、文件读取、对话历史）
4. 压缩后的请求转发至华为云 MaaS API
5. MaaS 路由至配置的 LLM 进行推理
6. 响应经 Headroom 返回至 OpenCode
7. 开发者在终端中看到响应

### 1.3 方案优势

- **60-95% Token 压缩**：Headroom 的上下文压缩引擎可大幅降低 Token 消耗，不影响回答质量
- **开源零许可费**：OpenCode 和 Headroom 均为开源项目，无供应商锁定
- **终端原生工作流**：OpenCode 直接在终端中运行，融入现有开发流程
- **多模型支持**：通过简单配置即可切换 DeepSeek、Claude 等模型

### 1.4 约束与限制

- 该解决方案部署前，需注册华为账号并开通华为云，完成实名认证，且账号不能处于欠费或冻结状态。
- 如果计费模式选择"包年包月"，请确保账户余额充足以便一键部署资源时自动支付；或者进入"费用中心 > 待支付订单"手动支付。
- 部署完成后需等待约 10 分钟，待 Headroom 代理和 OpenCode 完成初始化后方可使用。
- 使用前需在华为云 ModelArts MaaS 获取 API Key 并配置环境变量。

## 3. 实施步骤

### 3.1 准备工作

#### 3.1.1 获取 MaaS API Key

步骤1 登录华为云控制台，进入"ModelArts ModelArts Studio"服务。

步骤2 在左侧导航栏中选择"MaaS > API Keys"。

步骤3 单击"创建 API Key"，填写名称后单击"确定"。

步骤4 复制生成的 API Key 并妥善保存，后续配置 Headroom 时需要用到。

----结束

#### 3.1.2 创建 rf_admin_trust 委托（可选）

步骤1 进入华为云官网，打开控制台管理界面，鼠标移动至个人账号处，打开"统一身份认证"菜单。

步骤2 进入"委托"菜单，搜索 `rf_admin_trust` 委托。

如果委托存在，则不用执行接下来的创建委托的步骤。
如果委托不存在时执行接下来的步骤创建委托。

步骤3 单击"创建委托"按钮，在委托名称中输入 `rf_admin_trust`，委托类型选择"云服务"，输入 `RFS`，单击"完成"。

步骤4 单击"立即授权"。

步骤5 在搜索框中输入 `Tenant Administrator` 并勾选搜索结果，单击"下一步"。

步骤6 选择"所有资源"，并单击"确定"完成配置。

步骤7 "委托"列表中出现 `rf_admin_trust` 委托则创建成功。

----结束

### 3.2 快速部署

> 本章节帮助用户高效地部署"Headroom + OpenCode"解决方案。一键部署该解决方案时，参照本章节中的步骤和说明进行操作，即可完成快速部署。

步骤1 单击"开始部署"，跳转至 RFS 部署页面。

步骤2 单击"下一步"，确认基础配置与设置 ECS 密码。

**表 3-1 基础配置说明**

| 参数 | 说明 | 默认值 |
|------|------|--------|
| solution_name | 解决方案名称，4-24个字符，支持小写字母、数字、-（中划线），必须以小写字母开头 | headroom-opencode |
| ecs_flavor | 云服务器实例规格 | 推荐 x1.2u.4g |
| ecs_password | 云服务器密码，8-26位，至少包含大写字母、小写字母、数字和特殊字符中的三种 | / |
| system_disk_size | 系统盘大小（GB），取值范围：40-1024 | 40 |
| bandwidth_size | 弹性公网带宽（Mbit/s），按流量计费，取值范围：1-300 | 300 |
| charging_mode | 计费模式：postPaid（按需）或 prePaid（包年包月） | postPaid |

步骤3 进行加密与权限设置，按需选择。随后单击"下一步"。

步骤4 最后确认资源栈内容，可根据"创建执行计划"后免费获取预估费用（可选），单击"直接部署资源栈"。

步骤5 等待 `Apply required resource success` 完成资源部署，单击"输出"查看连接信息，等待约 10 分钟后开始使用服务。

> **说明：**
> - 创建新账户或账户余额不足时，请进入"费用中心 > 充值"页面进行充值。
> - 若选择了包年/包月计费模式，因账户余额不足导致自动支付失败，请进入"费用中心 > 待支付订单"页面，手动完成费用支付。

----结束

### 3.3 开始使用

#### 3.3.1 配置 API Key 并启动 OpenCode

步骤1 通过 SSH 连接到部署完成的 ECS 实例（连接信息可在 RFS 输出中查看）。

步骤2 设置 MaaS API Key：

```bash
export ANTHROPIC_AUTH_TOKEN='your-maas-api-key'
source /root/.bashrc
```

> **说明：** 将 `your-maas-api-key` 替换为步骤 3.1.1 中获取的实际 API Key。

步骤3 启动 OpenCode：

```bash
opencode
```

步骤4 验证 Headroom 代理状态：

**表 3-2 Headroom 管理接口**

| 接口 | 说明 | 示例 |
|------|------|------|
| Health Check | 健康检查 | `curl http://localhost:8787/readyz` |
| Real-time Stats | 实时统计 | `curl http://localhost:8787/stats` |
| Metrics | Prometheus 指标 | `curl http://localhost:8787/metrics` |

步骤5 开始使用 OpenCode 进行 AI 辅助编程。Headroom 代理会自动压缩上下文，降低 Token 消耗。

----结束

> **注意：** 首次启动 OpenCode 时会自动创建配置文件 `/root/.config/opencode/opencode.json`，其中已预配置 `baseURL` 指向 Headroom 代理地址。

### 3.4 快速卸载

步骤1 登录资源编排 RFS 资源栈，找到该解决方案创建的资源栈，单击资源栈名称右侧"删除"按钮。

步骤2 在弹出的删除资源栈确定框中，删除方式选择"删除资源"，输入 `Delete`，单击"确定"，即可卸载解决方案。

----结束
