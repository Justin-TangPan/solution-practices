# AiToEarn，AI 赋能的内容营销平台 — 一键部署

## 方案概述

### 应用场景

[AiToEarn](https://github.com/yikart/AiToEarn) 是一个 AI 驱动的内容营销平台，帮助创作者、品牌和企业在 13+ 国内外社交媒体平台上实现内容创作、分发、互动和变现。

核心能力：
- **多平台发布** — 一键发布到抖音、小红书、快手、B站、视频号、微信公众号等
- **互动自动化** — 自动点赞、收藏、关注，AI 智能评论回复
- **内容创作** — AI 视频生成、图片生成、批量矩阵运营
- **变现任务** — CPS/CPE/CPM 三种结算模式

### 方案架构

```
ECS（华北-北京四，8vCPU 16GB）
├── Docker Compose（12 个容器）
│   ├── MongoDB          ← 数据库
│   ├── Redis            ← 缓存
│   ├── RustFS           ← S3 兼容存储
│   ├── aitoearn-ai      ← AI 服务
│   ├── aitoearn-server  ← 后端服务
│   ├── aitoearn-web     ← 前端
│   ├── Nginx            ← 反向代理
│   └── 初始化容器 ×5
└── 端口: 8080 (Web)
```

### 方案优势

- **一键部署** — 12 个容器自动编排，开箱即用
- **全平台覆盖** — 支持 13+ 国内外主流社交媒体平台
- **AI 驱动** — 内置多种 AI 模型，支持视频/图片生成、智能评论
- **自托管** — 数据完全在自己的服务器上，隐私可控

### 约束与限制

- 需要 8vCPU 16GB 以上的 ECS 规格（12 个容器较吃资源）
- 各平台 OAuth 凭证需要用户自行申请
- AI 功能需要配置对应的 API Key

## 准备工作

- 注册华为云账号，完成实名认证
- 账户不得处于欠费或冻结状态

## 快速部署

1. 登录华为云控制台 → RFS（资源编排）→ 创建资源栈
2. 上传 `deploying-aitoearn.tf`
3. 填写参数（ECS 密码必填）
4. 点击创建，等待约 10-15 分钟

## 开始使用

部署完成后，浏览器访问 `http://YOUR_EIP:8080`。

### 配置社交媒体账号

1. 登录 AiToEarn 后台
2. 进入"账号管理"，添加要管理的社交媒体账号
3. 按提示完成 OAuth 授权

### 配置 AI 功能（可选）

如需使用 AI 内容创作功能，需要在 `/opt/aitoearn/` 目录下配置对应的 API Key。

## 快速卸载

1. 登录 RFS 控制台
2. 找到资源栈 → 删除 → 删除资源 → 输入 `Delete` → 确定
